# 環境構築

## ① Voltaのインストール
-  node.jsのバージョンが合わなくなると不具合が出る可能性があるのでvoltaを使ってバージョンを揃えるようにします。
-  バージョンは`18.15.0`です
- [こちらを参考にお願いします。](https://zenn.dev/taichifukumoto/articles/how-to-use-volta)

## ② npmパッケージをインストールする
- `npm install`

## ③ サーバーを立ち上げる
- cd プロジェクトファイル名 →　`npm run dev` で　http://localhost:3000/ にアクセス
- 問題無く開けていることを確認
